# Miro Clone v1

Primeira versão do widget de lousa interativa.

## Funcionalidades
- Gestão de Quadros (CRUD via localStorage).
- Ferramentas da lousa: Seleção/Mover, Caneta (desenho livre), Formas geométricas, Post-its, Texto.
- Zoom e arrasto na tela.
- Sem o uso de `<canvas>`, construído inteiramente com HTML/CSS (Shadow DOM) e SVG.
- Build final gera um único arquivo JS independente.
