# Notas macOS Widget

Este é um widget de bloco de notas inspirado no macOS, projetado para oferecer uma experiência de anotações rica e elegante.

## Recursos
- **Estilo macOS:** Interface limpa, cantos arredondados, fontes do sistema e tipografia cuidadosa.
- **Suporte a Markdown:** Escreva e visualize notas em Markdown de forma integrada.
- **Duas Colunas:** Gerencie suas notas na barra lateral enquanto edita no painel principal.
- **Armazenamento Local:** As notas são armazenadas com segurança no navegador do usuário usando `localStorage`.

## Como usar
Este widget é encapsulado como um Web Component (`<mac-notes-widget>`) usando Shadow DOM. Basta importar o arquivo `dist/build.js` e usar a tag em seu HTML.
