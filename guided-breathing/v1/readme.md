# Guided Breathing Widget v1

## Features

- **Catalog of Exercises:** Square Breathing, 4-7-8, Diaphragmatic.
- **Focus Mode:** Distraction-free training area.
- **Visual Guide:** Smooth animations for breathing phases.
- **Audio Guide:** Synthesized tones via Web Audio API.
- **Customizable:** Adjustable times and sound toggle.
- **Modern UI:** Built as a Web Component with Shadow DOM. No external dependencies, single injected JS file.
