# Teclado Sintetizador V1
Versão 1 do widget de teclado sintetizador configurável para afinação vocal.
Recursos:
- Teclado visual com notas/cifras
- Quantidade de oitavas configurável (1 a 3)
- Múltiplos timbres (Piano, Órgão)
- Som gerado via Web Audio API (muito leve)
- Salva preferências do usuário no LocalStorage
- Web Component autossuficiente (Shadow DOM)
