# Pomodoro Timer

Um elegante widget de Pomodoro utilizando Web Components, Shadow DOM e Vanilla JS.

## Funcionalidades
- Timer com ciclos de Foco, Pausa Curta e Pausa Longa.
- Interface moderna, limpa e com animações suaves.
- Armazenamento de dados do usuário localmente (LocalStorage).
- Ocupa perfeitamente o espaço disponível onde é injetado.

## Como Usar
Ao incorporar em sua página, o widget ajusta o seu tamanho ao elemento pai automaticamente.
