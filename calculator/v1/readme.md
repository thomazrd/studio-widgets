# Calculator Widget v1

## Overview
Version 1 of the Calculator Widget introduces the core functionality for both Standard and Scientific modes.

## Highlights
- Encapsulated Shadow DOM layout to prevent CSS leakage.
- Clean and premium Apple-like UI with smooth transitions and interactions.
- Evaluates mathematical expressions securely using a custom state/logic handler.
- Responsive design tailored to fit exactly the dimensions of the parent container.
- Error prevention and handling explicitly designed.
