# Calculadora Widget

Uma calculadora profissional embutível como Web Component.
Oferece múltiplas visualizações (Light, Dark, Neon) e armazenamento local de histórico com prefixo seguro via UUID.

- Sem o uso de `<canvas>`.
- `width: 100%`, `height: 100%`, sem margens externas e bordas.
- Customizável via Shadow DOM.
