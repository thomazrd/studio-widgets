A Calculadora Profissional é um widget embeddable desenhado para alta performance e excelente UI/UX.
Desenvolvida usando as mais modernas especificações de Web Components e Shadow DOM, ela garante que os estilos não vão vazar ou quebrar com os da página hospedeira.

Principais funcionalidades:
- **Design Responsivo**: Ocupa sempre 100% do elemento container hospedeiro.
- **Múltiplos Temas**: Alternância suave entre temas Light, Dark e Neon.
- **Armazenamento de Histórico**: Salva as últimas operações no `localStorage` do navegador de forma isolada, não interferindo com outros widgets, utilizando seu UUID nativo como prefixo de chave de armazenamento.
- **Animações Fluidas**: Feedbacks de clique satisfatórios e transições suaves entre temas.

Ideal para qualquer aplicação ou site que necessite de uma calculadora elegante que apenas funciona.
